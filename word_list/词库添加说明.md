# 添加和切换单词表

网页默认读取当前目录中的：

`SAT词汇词根 联想记忆法(乱序版) (俞敏洪).json`

## 方法一：在网页中导入

点击网页里的“导入 JSON”，可以一次选择一个或多个词库。导入后的词库保存在浏览器本地，并会出现在词库下拉框中。

## 方法二：把 JSON 放入 word_list 文件夹

把新 JSON 放进 `word_list`，然后点击网页里的“扫描词表”。如果部署环境不允许读取目录列表，请同时把文件加入 `wordbooks.json`：

```json
{
  "books": [
    {
      "name": "我的新词库",
      "file": "my_new_wordbook.json"
    }
  ]
}
```

## 支持的数据格式

当前 SAT JSON 的 `words + senses` 格式可以直接使用，也支持简单数组：

```json
{
  "title": "我的词库",
  "words": [
    {
      "word": "lucid",
      "phonetic": "/ˈluːsɪd/",
      "meaning": "清晰的；易懂的",
      "example": "His explanation was lucid.",
      "memory": "luc 表示光，有光就清楚。"
    }
  ]
}
```

字段也兼容 `term`、`english`、`translation`、`definition`、`chinese`、`ipa`、`pronunciation`、`sentence` 等常见命名。

每个词库会分别保存熟悉度、错词和学习顺序，切换词库不会混用进度。

## 导出与恢复学习数据

学习进度和用户导入的词库默认保存在当前浏览器中。为了防止清除浏览器数据、换设备或系统重装导致记录丢失，请定期使用页面词库栏中的“导出备份”。

导出的 JSON 包含每日计划、熟悉度、错词、复习偏好、设置和所有通过“导入 JSON”添加的自定义词库。需要恢复时，点击“恢复备份”并选择该 JSON；恢复会替换当前学习进度，同时保留并合并当前浏览器已有的自定义词库。
